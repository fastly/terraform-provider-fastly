# fastly_domains_v1

Use this data source to get information about Fastly domains.

## Example Usage

```terraform

}

  output "all_domains" {
  value = data.fastly_domains_v1.example.domains
}

output "total_domains" {
  value = data.fastly_domains_v1.example.total
  
}
```
