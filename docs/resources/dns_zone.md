# fastly_dns_zone

Provides a Fastly DNS Zone.

## Example Usage

```terraform
resource "fastly_dns_zone" "example" {
    name        = "example.com."
    description = "My secondary DNS zone"

    xfr_config_inbound {
      inbound_tsig_key_id = fastly_tsig_key.example.id

      primaries {
        address     = "192.0.2.1"
        description = "Primary nameserver"
      }

      primaries {
        address     = "192.0.2.2"
        description = "Secondary primary nameserver"
      }
    }
  }
```

## Import

Fastly DNS Zones can be imported using their Zone ID, e.g.

```sh
$ terraform import fastly_dns_zone.example xxxxxxxxxxxxxxxxxxxx
```
